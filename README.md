### Wolowitz

A simple HTML5 video player.

_Install_

Just include the js and css files in your document.
To include videos in your code:
```html
<div class="ww-element" data-ww-element= "true" src="video.mp4"> </div>
```
Then initialize Wolowitz:
```js
  new Wolowitz({
    selector: "#video"
  });
```
Done!
